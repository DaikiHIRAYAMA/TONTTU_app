## 開発言語
<br>
  ・Ruby3.0.1
  ・Rails6.0.5
  
## 実行手順
 ```bash
 $ git clone https://github.com/DaikiHIRAYAMA/TONTTU.git
 $ cd TONTTU
 $ bundle install
 $ rails db:create && db:migrate
 $ rails db:seed

 ```
